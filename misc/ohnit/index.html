소개​

온잇은 커플이 각자 업로드한 일상 사진을 AI가 실시간으로 분석하여 비슷한 순간을 자동으로 매칭해주는 메신저 앱입니다. 서로 다른 시간과 장소에서 찍은 사진들 속에서 의미 있는 연결고리를 찾아내어, 커플의 특별한 순간들을 함께 공유할 수 있습니다.

핵심 기능​

AI 사진 매칭​

각자 올린 사진을 AI가 분석하여 시각적으로 유사하거나 의미상 연관된 사진들을 자동으로 찾아 매칭합니다. 같은 장소, 비슷한 날씨, 유사한 분위기의 사진들이 자연스럽게 연결됩니다.

실시간 알림​

상대방의 사진과 매칭이 발생하면 즉시 푸시 알림으로 알려드립니다. 일상 속에서 우연히 발견하는 공통점이 특별한 대화의 시작이 됩니다.

인생네컷 스타일 공유​

매칭된 사진들을 인생네컷 스타일의 예쁜 이미지로 저장하고 공유할 수 있습니다. 두 사람의 순간을 하나의 추억으로 간직하세요.

시간순 정렬​

사진의 촬영 시간을 자동으로 인식하여 시간 순서대로 정리합니다. 커플의 일상이 시간의 흐름 속에서 자연스럽게 이어집니다.

기술적 특징​

하이브리드 AI 시스템​

  • CLIP 모델을 활용한 시각적 유사도 분석
  • Florence-2 모델을 통한 장면 이해
  • Qwen2.5 모델로 한국어 설명 생성
  • GPU 가속으로 0.5초 이내 실시간 처리

개인정보 보호​

  • 커플별로 완전히 격리된 데이터 저장
  • Firebase 인증 기반 보안 시스템
  • 카카오 소셜 로그인 지원

안정적인 서비스​

  • React Native 기반 iOS/Android 지원
  • 실시간 동기화 및 백업 시스템
  • Docker 기반 안정적인 배포 환경
